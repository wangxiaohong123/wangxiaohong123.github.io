redis安装

发表于 2020-11-21 | 分类于 数据库

单机安装

将redis-3.2.8.tar.gz传到虚拟机/usr/local/下，执行命令tar -zxvf redis-3.2.8.tar.gz解压，然后cd到redis-3.2.8目录下执行make && make test。

安装的时候报错，通过错误看到make已经执行完了，在make test的时候报缺少tcl错误，所以要安装一下tcl。

执行命令安装tcl

安装完tcl之后进入reddis目录继续执行make test，make test时可能会报错，没说是啥错就没管。

make test之后执行make install。

redis cli的使用

redis-cli SHUTDOWN，连接本机的6379端口停止redis进程

redis-cli -h 127.0.0.1 -p 6379 SHUTDOWN，指定要连接的ip和端口号

redis-cli PING，ping redis的端口，看是否正常

redis-cli，进入交互式命令行

进入redis3.2.8目录下的src执行./redis-benchmark -h 192.168.0.5进行压测
